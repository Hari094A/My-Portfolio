# Personal Portfolio Website

This is a responsive personal portfolio website built using HTML5, CSS3, and JavaScript. The website showcases my profile, skills, projects, and contact information with a clean and modern user interface.

Features:
- Responsive design for mobile, tablet, and desktop
- Light and dark mode theme toggle
- Modern CSS styling
- CSS Grid and Flexbox layouts
- Smooth hover effects and animations
- Sections for About, Skills, Projects, and Contact

Technologies Used:
- HTML5
- CSS3
- JavaScript

Project Description:
This portfolio website represents my frontend development skills and demonstrates my ability to create responsive and user-friendly web interfaces using modern web technologies.
